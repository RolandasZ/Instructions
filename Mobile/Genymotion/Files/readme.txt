
For testing android emulator download genymotion from https://www.genymotion.com/.

Run device

1 Step - Drag ARM emulator file to phone desktop and flash it:
Android 4.4.4, 5.0 - Genymotion-ARM-Translation_v1.1.zip
Android 5.1, 6.0 - ARM_Translation_Lollipop_20160402.zip

2 Step - Drag Gapps file to phone desktop and flash it:
Android 4.4.4 - gapps-kk-20140606-signed.zip
Android 5.0 - gapps-lp-20141109-signed.zip
Android 5.1 - open_gapps-x86-5.1-pico-20161122
Android 6.0 - benzo-gapps-M-20151011-signed-chroma-r3.zip. Sign in to goole. then gapps-L-4-21-15.zip
Android 7.0

gapps:
http://opengapps.org/ (Use 'Pico' - minimum to get google play functionality)

Full documentation here:
https://github.com/codepath/android_guides/wiki/Genymotion-2.0-Emulators-with-Google-Play-support

----

When gapps is installed, google+ may crash all the time.
Just restart the device and disable goolge+
